# VELOCUE Hero 文案選項 v1

最後更新：2026-03-19
適用範圍：VELOCUE 官網首頁 Hero
用途：收斂主標 / 副標 / 輔助短句方向，供後續設計與實作選用

## 1. 使用方式

這份文件不是要一次決定唯一答案。
它的目的，是把 Hero 文案拆成不同策略方向，方便比較：

- 哪一組最能代表 VELOCUE
- 哪一組最有記憶點
- 哪一組最適合現在的首頁

目前所有版本都以前提為準：

- 主承諾是提升騎前準備品質
- 產品是 route briefing tool，不只是列印工具
- 語氣要原生繁中、冷靜、克制

## 2. 建議結論先說

如果現在要先選一組進首頁，我建議：

- 主標：`把騎前準備，直接做進路線裡。`
- 副標：`從地圖與等高線出發，把補給、會合、配力與提醒整理成真正帶得上路的騎乘筆記。`
- 輔助短句：`行前小抄，坡前不慌`

理由：

- 主標穩，能撐住產品定位
- 副標清楚，能接住方法與場景
- 輔助短句有記憶點，能補上在地口語感

## 3. Option A：定位最完整

### 主標

把騎前準備，直接做進路線裡。

### 副標

從地圖與等高線出發，把補給、會合、配力與提醒整理成真正帶得上路的騎乘筆記。

### 輔助短句

行前小抄，坡前不慌

### 特性

- 最穩
- 最完整
- 最能代表 VELOCUE 上位概念
- 適合當首頁正式版本

### 風險

- 記憶點不如口語型版本強

## 4. Option B：口語記憶點強

### 主標

行前小抄，坡前不慌

### 副標

把補給、會合、配力與提醒先整理進路線裡，出發前看得懂，路上也看得到。

### 輔助短句

給公路車、耐力與比賽情境的騎前準備工具

### 特性

- 最有記憶點
- 最有在地語感
- 第一眼比較有 personality

### 風險

- 主標承載力較窄
- 偏爬坡情境
- `小抄` 一詞會讓成熟度略降

### 適合用法

- Hero 替代版測試
- 社群宣傳主句
- Landing page campaign 版本

## 5. Option C：產品定義最清楚

### 主標

不只看路線，也把準備一起整理好。

### 副標

VELOCUE 讓你在出發前對照地圖與等高線，把補給、提醒與輸出整理成同一套騎乘筆記。

### 輔助短句

把路線變成真的用得上的準備

### 特性

- 產品理解快
- 對第一次接觸的人友善
- 好接功能示意

### 風險

- 記憶點普通
- 品牌感較弱

## 6. Option D：情境感較強

### 主標

在出發前，把重要安排先放進路線。

### 副標

不是只知道怎麼走，而是先知道哪裡該補給、哪裡該保留節奏，哪些事不能留到路上才想。

### 輔助短句

有些準備，應該在上車前就先整理好

### 特性

- 情境感好
- 很接近品牌語氣
- 適合搭配強視覺 Hero

### 風險

- 產品名稱與方法需要副標補足

## 7. Option E：偏成熟 editorial

### 主標

重要的事，最好先留在路線裡。

### 副標

從補給到爬坡提醒，把真正會影響騎乘判斷的資訊，整理成一條更清楚的騎前筆記。

### 輔助短句

給需要準備的人，不只是給看路線的人

### 特性

- 比較成熟
- 比較有品牌頁質感
- 適合做高留白版面

### 風險

- 抽象度稍高
- 若視覺證據不夠，可能不夠像工具網站

## 8. 不建議作為主標的方向

以下方向可以做輔助文案，但不建議做 Hero 主標：

- 提升你的騎乘表現
- 更完整的賽事策略
- 路線、海拔、筆記，一站整理
- 把 GPX 變成 cue strip

原因：

- 太像運動品牌或功能摘要
- 要嘛過度抽象，要嘛過度狹窄

## 9. 判斷標準

Hero 版本選擇時，請優先檢查：

1. 能不能一眼看出這是騎前準備工具？
2. 能不能承載 VELOCUE 未來比 cue strip 更大的定位？
3. 有沒有在地語感，但不過度口語？
4. 放進高留白 Hero 後，還站得住？
5. 與產品畫面放在一起時，會不會太像 slogan 海報？

## 10. 目前建議排序

以「適合現在首頁正式版本」排序：

1. Option A
2. Option D
3. Option C
4. Option B
5. Option E

以「記憶點強度」排序：

1. Option B
2. Option A
3. Option D
4. Option E
5. Option C

## 11. 下一步建議

如果你同意目前方向，下一步最合理的是：

1. 選定一組 Hero 文案
2. 依這組 Hero 去做首頁視覺稿或前端首頁第一版

如果還想再收斂，可以再做一輪：

- 更成熟版
- 更口語版
- 更比賽導向版

三種語氣分支比較。
